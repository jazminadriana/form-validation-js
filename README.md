# Calorie Counter App

A dynamic web application that calculates daily calorie intake based on user inputs, featuring real-time validation, UI feedback, and responsive design.

## Live Demo

https://jazminadriana.github.io/form-validation-js/

---

## Project Overview

This project focuses on implementing dynamic form validation, business logic processing, and interactive UI updates using vanilla JavaScript.

The application calculates daily caloric needs based on user-provided data and provides immediate visual feedback through validation messages and a progress indicator.

---

## Core Features

- Real-time input validation
- Dynamic form field generation
- Business logic for calorie calculation
- Visual progress bar feedback
- Dark / Light mode toggle
- Responsive layout for mobile and desktop
- DOM manipulation for dynamic content updates

---

## Tech Stack

- HTML5
- CSS3 (Responsive + Variables)
- JavaScript (Vanilla JS)

## Responsive Design

Optimized for mobile and desktop devices.
