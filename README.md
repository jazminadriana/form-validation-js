# Calorie Counter Web App

Interactive web application to calculate daily calories with dynamic form validation.

## Features

- Real-time input validation
- Dark / Light mode toggle
- Visual progress bar
- Responsive design
- Advanced DOM manipulation

## Technologies

- HTML5
- CSS3 (Responsive + Variables)
- JavaScript (Vanilla JS)

## Responsive Design

Optimized for mobile and desktop devices.

## Objective

Project developed to strengthen skills in form validation, business logic, and user experience.
